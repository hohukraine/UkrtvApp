# Cursor AI - Промпт для роботи з проєктом УкрТБ

## Огляд проєкту
Це Android-додаток для перегляду фільмів/серіалів на Android TV з використанням Jetpack Compose, Media3 ExoPlayer та HLS/DASH стрімінгу.

## КОНТЕКСТ ДЛЯ ТЕСТУВАННЯ

### Тестові команди:
- **Build**: `./gradlew assembleDebug`
- **Release build**: `./gradlew assembleRelease`
- **Tests**: `./gradlew connectedDebugAndroidTest`
- **Lint**: `./gradlew lint` (перевірити, чи існує)
- **Format**: використовувати ktlint (якщо налаштовано)

### Бібліотеки:
- Media3 ExoPlayer 1.3.1 (HLS, DASH)
- Jetpack Compose (UI)
- Hilt (DI)
- DataStore (preferences)
- OkHttp 4.12.0
- Retrofit 2.11.0

## ПРИКЛАДИ ВИКОРИСТАННЯ

### Приклад 1: Аналіз відеоплеєра
```
Проаналізуй відеоплеєр. Як працює:
- Ініціалізація ExoPlayer
- Завантаження HLS/DASH потоків
- Керування буфером
- Обробка помилок відтворення
- Управління життєвим циклом
```

### Приклад 2: Оптимізація буферу
```
Як налаштувати буфери в ExoPlayer як у додатку "Ліфт":
- minBufferMs = 30000
- maxBufferMs = 120000
- bufferForPlaybackMs = 2500
- bufferForPlaybackAfterRebufferMs = 5000
```

### Приклад 3: Codec fallback
```
Як реалізувати fallback між якістями відео:
1. Спочатку спробувати HLS (h264)
2. Якщо помилка - переключитися на DASH VP9
3. Якщо помилка - переключитися на DASH AV1
4. Логувати кожну спробу
```
